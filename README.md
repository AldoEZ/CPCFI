# CPCFI Competitive Programming Repository

A curated collection of **competitive programming solutions**, notes, and reference implementations developed as part of my practice and participation in **CPCFI (Club de Programación Competitiva de la Facultad de Ingeniería)**.

This repository contains solved problems from platforms such as **Codeforces**, **CSES**, and other online judges, along with implementations of common algorithms and data structures used in programming contests.

---

## About

Competitive programming is a discipline focused on solving algorithmic problems under time and memory constraints.  
This repository was created to organize solutions, practice material, and reusable techniques that are useful for contests, training sessions, and technical growth.

It reflects work related to:

- Personal contest practice
- Problem solving training
- Algorithmic study
- Reference code for common techniques
- Preparation for programming competitions

---

## Platforms Covered

Problems and exercises in this repository may come from:

- **Codeforces**
- **CSES Problem Set**
- **OmegaUp**
- **Kattis**
- **ICPC-style problem sets**
- Other online judges and training resources
